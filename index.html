Local Demo (Dev)

  1. Install deps: ffmpeg, Python 3.10+, Node 18+.
  2. In the repo, run DAVID/dev-setup-windowed.sh (or DAVID/board/start.sh for the UI + API).
  3. Open the board at http://localhost:3000.
